# Privacy

This plugin sends the content you pass to it, plus any metadata you set (source,
external_id, callback_url), to your configured LoopQuest deployment over HTTPS so
a human can review it. Your API key is stored by Dify as a secret credential and
sent only to LoopQuest as a Bearer token. The plugin stores no data itself.

See the LoopQuest privacy policy at `https://<your-app>/privacy`.
